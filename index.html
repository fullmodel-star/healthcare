資料會送到哪裡？
• 所有健康紀錄只存在「你這台裝置的瀏覽器」，預設不上傳任何地方。
• 只有當你主動按「拍照辨識」時，那張餐盤照片會傳到 Google Gemini 辨識。
• 只有當你主動按「AI 建議」時，會把當天的體重/BMI/睡眠/飲水等摘要傳到 Google。
• 不按這兩個功能、或開啟上面的「離線模式」，就完全不會連網外傳。
Google Gemini API Key 啟用 AI 功能（選填）
至 Google AI Studio 免費取得。Key 僅儲存於本機 localStorage。請勿在公用電腦使用。不填也能用搜尋／手動記錄。
資料管理
資料只存在本機瀏覽器；清除瀏覽器網站資料會一起刪掉，請定期「匯出備份」。清除時會詢問是否一併清掉 API Key。
